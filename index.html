Bóc tách dữ liệu ở tập tin DOCX thành CSV từ điển

Công dụng:

Chương trình này giúp phân tích một file DOCX, ví dụ nội dung một từ điển.
Người dùng cho biết bố cục từ điển: tiếng Đức có định dạng gì, tiếng Anh định dạng gì, khi nào thì kết thúc một mục từ.
Chương trình bóc tách từ file DOCX để ra các cặp nội dung tiếng Đức và tiếng Anh tương ứng, rồi xuất ra file CSV song ngữ (mở được bằng Excel, dùng được trong CAT).

Hướng dẫn:

  + Chọn file văn bản .DOCX để upload.
  + Chọn ngôn ngữ nguồn, ngôn ngữ đích, cùng định dạng của chúng trong văn bản.
  + Chọn điều kiện kết thúc một mục từ.

IMPORTANT: Bạn có muốn XÓA các file tải lên sau khi xử lý xong? (kết quả vẫn sẽ được lưu trên trang web này)
(default) Có Không

DOCX file:
Ngôn ngữ nguồn:

Định dạng ứng với ngôn ngữ nguồn: đậm hay không?   nghiêng hay không?

Ngôn ngữ đích:

Định dạng ứng với ngôn ngữ đích: đậm hay không?   nghiêng hay không?

Điều kiện kết thúc một mục từ:

End-of-line ⏎ (xuống dòng)   bold->not bold (hết đậm)   not bold-> bold (bắt đầu đậm)   italic->not italic (hết nghiêng)   not italic-> italic (bắt đầu nghiêng)
